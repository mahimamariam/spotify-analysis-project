# Spotify Songs Analysis

## Description
This project is a web application developed using Node.js and MySQL to manage and analyze Spotify song data.

## Features
- Add, update, delete songs (CRUD)
- Dashboard with analytics
- Top songs visualization
- Search functionality

## Technologies Used
- Node.js
- MySQL
- HTML, CSS, JavaScript

## How to Run
1. Install Node.js
2. Run server:
   node server.js
3. Open browser:
   http://localhost:3000
